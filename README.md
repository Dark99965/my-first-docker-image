# my first docker image

Hello, world! This is my first docker image and working with docker     

## IMPORTENT!

- **The project is set up with pnpm so we recommend to use pnpm to set up the project**     

- **This project requises Node.js and npm and will not run without it**

## How to setup

First install the deps with your **package manger**     
example:
~~~sh

npm install

~~~

Next run this cmd to set up the dev server

~~~sh

npm run dev

~~~
